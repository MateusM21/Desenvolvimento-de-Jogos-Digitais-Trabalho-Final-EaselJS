 Flappy 

Um jogo de navegador estilo arcade inspirado no clássico "Flappy Bird", desenvolvido com foco em mecânicas de física simples, coleta de itens e progressão de dificuldade.

 Descrição Geral

 Enredo
Você controla "Flappy", um pássaro determinado a voar o mais longe possível através de um cenário infinito de canos verdes e nuvens. Não há fim, apenas a busca pela perfeição e pela maior pontuação.

 Objetivos
1.Sobreviver: Voe entre os vãos dos canos sem tocar neles ou no chão.
2.Pontuar: Cada par de canos ultrapassado vale 1 ponto na pontuação principal.
3.Coletar: Pegue as Moedas de Ouro giratórias que aparecem aleatoriamente (35% de chance) entre os canos para aumentar seu contador de tesouros.
4.Recorde: O jogo salva automaticamente sua melhor pontuação (`High Score`) no navegador. Tente batê-la!

Ferramentas Utilizadas

Este projeto foi construído utilizando tecnologias Web padrão e bibliotecas de manipulação gráfica:

HTML5 & CSS3: Estrutura da página e estilização dos menus (com fonte *Press Start 2P* para estética Pixel Art).
JavaScript (ES5/ES6): Lógica do jogo.
[CreateJS](https://createjs.com/): Suíte de bibliotecas para facilitar o desenvolvimento de jogos em Canvas:
EaselJS: Manipulação de sprites, imagens e o palco (Stage).
TweenJS: Animações suaves (movimento dos canos e rotação do pássaro).
SoundJS / HTML5 Audio: Gerenciamento de efeitos sonoros e música.
PreloadJS: Carregamento eficiente de assets.

Funcionalidades

* **Menu Completo:** Iniciar Jogo, Opções, Placar e Game Over.
* **Sistema de Dificuldade:** Opções para velocidade "Normal" ou "Rápido".
* **Controle de Áudio:** Sliders independentes para volume de Efeitos (SFX) e Música (BGM), com opção de Mudo.
* **Persistência de Dados:** O recorde é salvo no `localStorage` do navegador.
* **Efeitos Visuais:** Partículas ao pular, coletar moedas e colidir, além de efeito de tremor de tela (Screen Shake).

 Instruções de Instalação e Execução

Devido às políticas de segurança dos navegadores modernos (CORS), este jogo **não funcionará corretamente** se você apenas clicar duas vezes no arquivo `index.html`. É necessário um servidor local.

### Pré-requisitos
* Um navegador moderno (Chrome, Firefox, Edge).
* Um editor de código (recomendado: **VS Code**).

### Passo a Passo

1.  **Baixe o Projeto:**
   
2.  **Executando com VS Code (Recomendado):**
    * Abra a pasta do projeto no **VS Code**.
    * Instale a extensão **"Live Server"**
    * Clique com o botão direito no arquivo `index.html`.
    * Selecione a opção **"Open with Live Server"**.
    * O jogo abrirá automaticamente no seu navegador padrão.

### Como Jogar
Mouse: Clique com o botão esquerdo na tela para fazer o pássaro voar.
